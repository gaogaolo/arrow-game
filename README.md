# 挪箭头 - 全功能开发版

一个基于网页的箭头消除游戏关卡生成器。

## 功能特点

- 🚀 **全自动铺满** - 自动生成完整的箭头关卡
- ✏️ **手动绘制** - 支持手动绘制箭头
- 🎨 **形状生成** - 支持文字/Emoji/SVG生成特殊形状的关卡
- 📥 **导入/导出** - 支持JSON格式的关卡配置导入导出
- 🔍 **关卡校验** - 验证关卡可解性
- 🚫 **禁区设置** - 可设置禁区格子

## 文件结构

## 使用方法

1. 直接用浏览器打开 `test7.html` 即可运行
2. 或部署到任意静态服务器

## 游戏玩法

- 点击箭头可以消除箭头
- 箭头指向的方向如果没有其他箭头阻挡，则可以消除
- 目标是消除所有箭头
